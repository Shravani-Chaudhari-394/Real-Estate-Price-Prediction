# Changelog

All notable changes to the Real Estate Price Prediction System.

## [1.2.0] - 2026-07-07

### Added
- Ensemble model combining XGBoost and Neural Network
- Batch prediction endpoint
- Prometheus metrics and Grafana dashboards
- CI/CD pipeline with GitHub Actions
- Comprehensive test suite (>80% coverage)

### Changed
- Improved API latency to <200ms average
- Updated frontend dashboard with real-time analytics

## [1.1.0] - 2026-06-15

### Added
- MLflow model registry integration
- Redis caching layer
- Authentication middleware

## [1.0.0] - 2026-06-01

### Added
- Initial FastAPI backend with prediction endpoints
- XGBoost ML pipeline
- React dashboard
- Docker Compose deployment
